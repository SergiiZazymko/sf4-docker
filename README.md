# sf4-docker

Simple docker container wich contains:

* Symfony4
* PHP 7.2
* Apache2
* MySQL
* Adminer
* Composer

Just run:

 * docker-compose up -d
 * docker-compose run composer install
 * http://localhost - Symfony4 application
 * http://localhost:8080 - Adminer
 * Database - user: root , password: 55555
